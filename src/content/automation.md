> 키워드 하나만 입력하면 영상 1개가 완성되기까지 **약 1분**.
> 그동안 당신이 직접 할 일은 **단 한 가지, 기획**뿐입니다.

## 자동화 흐름 한눈에 보기

```
키워드 입력
   ↓
상품 검색 (쿠파스 자동)
   ↓
AI 스크립트 + 후킹 카피 생성
   ↓
영상 소스 자동 매칭
   ↓
자막 자동 정렬 (오타·단위 보정)
   ↓
수익화 링크 자동 연동
   ↓
완성 영상 출력 (.mp4)
```

## 자동화로 절약되는 시간

| 작업 | 수동 | 크로닛 자동 |
|------|------|-------------|
| 키워드 리서치 | 20분 | 즉시 |
| 스크립트 작성 | 30분 | 10초 |
| 영상 소스 수집 | 15분 | 즉시 |
| 자막 입력·정렬 | 25분 | 5초 |
| 수익화 링크 삽입 | 5분 | 즉시 |
| **합계** | **약 95분** | **약 1분** |

> 영상 1개당 평균 **94분 절약**. 하루 5개만 만들어도 8시간 절약.
