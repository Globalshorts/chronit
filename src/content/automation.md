> 쇼핑 릴스 링크만 입력하면 영상 1개가 완성되기까지 **약 2분**.
> 그동안 당신이 직접 할 일은 **없습니다.**
(단, 1회 3, 4, 5 단계에서 원하시는 목소리, 자막 스타일, 썸네일 스타일 설정해두면 퀄리티를 올릴 수 있습니다)

## 자동화 흐름 한눈에 보기

```
쇼핑 숏폼 링크 입력
   ↓
영상 담기
   ↓
자동 실행 버튼 클릭
   ↓
완성 영상 출력 (.mp4)
```

## 1. 링크 → 콘텐츠 자동 변환

링크 입력만으로 클릭 몇 번이면 영상 생성 가능

## 2. AI 스크립트 + 후킹 자동 생성

- **0.1초 후킹 카피** — 시청자가 스크롤을 멈추게 만드는 첫 문장 자동 생성
- **자막 절단 알고리즘** — 말 끊김 없는 자연스러운 자막 배치
- **스토리텔링 체인** — 결말까지 끌고 가는 문장 구조 자동 적용

## 3. 자막 + 소스 자동 정렬

- 오타 자동 보정
- 자막과 음성의 싱크 맞춤 97%
- 컷 전환 타이밍에 맞춘 자막 출현
- 저작권 안전한 영상 소스 자동 매칭

## 4. 수익화 링크 자동 연동(상품 검색 탭)

- **쿠파스** 상품 자동 검색 → 자동 매칭
- **인포크링크** 즉시 접속 가능

## 5. 스타일 찾기 가능

- 여러 쇼핑 유튜버의 스타일 카피 가능
- 채널별 톤/스타일 템플릿 자동 적용

## 자동화로 절약되는 시간

| 작업 | 수동 | 크로닛 자동 |
|------|------|-------------|
| 키워드 리서치 | 10분 | 10초 |
| 스크립트 작성 | 10분 | 10초 |
| 영상 소스 수집 | 30분 | 30초 |
| 자막 입력·정렬 | 20분 | 10초 |
| 전체 편집 | 15분 | 1분 |
| 상품 찾기 | 5분 | 즉시 |
| **합계** | **약 90분** | **약 2분** |

> 영상 1개당 평균 **88분 절약**. 하루 5개만 만들어도 7시간 절약.
