## 1단계: 회원가입 및 결제

1. 홈페이지에서 원하는 요금제(스타터/프로/마스터) 선택
2. 결제 모달에서 **계좌이체** 진행
3. 입금자명을 **가입 이메일과 동일하게** 입력 (자동 매칭용)
4. 영업일 기준 1일 이내 자동 활성화

## 2단계: 로그인

크로닛 데스크탑 앱을 실행하고 **결제 시 사용한 이메일(구글/카카오)**로 로그인합니다.

![](https://oxygqtbdpnxxcgzwdlzi.supabase.co/storage/v1/object/public/manual-images/login.png)

## 3단계: 스타일 찾기

영상을 만들기 전에 원하는 스타일을 먼저 찾아두세요.
좌측 메뉴의 **스타일 찾기** 탭에서 여러 쇼핑 유튜버의 스타일을 미리보기 할 수 있습니다.

![](https://oxygqtbdpnxxcgzwdlzi.supabase.co/storage/v1/object/public/manual-images/style-finder.png)

## 4단계: 새 프로젝트 생성

대시보드에서 **새 프로젝트** 버튼을 클릭합니다.

![](https://oxygqtbdpnxxcgzwdlzi.supabase.co/storage/v1/object/public/manual-images/dashboard.png)

## 5단계: 영상 담기

만들고 싶은 상품의 쇼핑 릴스/쇼츠 링크를 입력하고, 쓰고 싶은 영상을 **2개 이상** 담아주세요.

![](https://oxygqtbdpnxxcgzwdlzi.supabase.co/storage/v1/object/public/manual-images/step2-link.png)

## 6단계: 스크립트 확인

AI가 후킹 카피 + 본문을 한국어로 자동 작성합니다. 내용을 확인하고 원하면 수정하세요.

![](https://oxygqtbdpnxxcgzwdlzi.supabase.co/storage/v1/object/public/manual-images/step3-script.png)

## 7단계: 자막 · 썸네일 · 목소리 스타일 설정

자막 스타일, 썸네일 스타일, 목소리를 원하는 대로 설정합니다.

![](https://oxygqtbdpnxxcgzwdlzi.supabase.co/storage/v1/object/public/manual-images/step4-subtitle.png)

![](https://oxygqtbdpnxxcgzwdlzi.supabase.co/storage/v1/object/public/manual-images/step5-voice.png)

> **중요: 각 단계에서 반드시 저장 버튼을 눌러주세요.**
> 한 번 저장하면 이후 자동 실행 시 동일한 스타일이 자동 적용됩니다.

## 8단계: 자동 실행

스타일 설정을 마쳤으면 자동 실행 버튼을 클릭합니다.
이후부터는 링크만 입력하면 저장된 스타일로 자동 완성됩니다.

![](https://oxygqtbdpnxxcgzwdlzi.supabase.co/storage/v1/object/public/manual-images/autorun1.png)

![](https://oxygqtbdpnxxcgzwdlzi.supabase.co/storage/v1/object/public/manual-images/autorun2.png)

## 9단계: 완성 파일 확인

영상이 완성되면 지정한 저장 폴더에 두 파일이 생성됩니다.

-  — 완성된 쇼핑 숏폼 영상
-  — 유튜브/틱톡 업로드용 제목·설명·해시태그 자동 생성

![](https://oxygqtbdpnxxcgzwdlzi.supabase.co/storage/v1/object/public/manual-images/6_SEO.png)

![](https://oxygqtbdpnxxcgzwdlzi.supabase.co/storage/v1/object/public/manual-images/Output.png)

## 10단계: 상품 링크 찾기

좌측 메뉴 **상품 검색** 탭에서 영상에 맞는 상품 링크를 찾아주세요.

- **쿠파스** — 릴스 링크 또는 프로젝트로 상품 자동 검색
- **인포크링크** — 원클릭 접속으로 링크 바로 붙여넣기
