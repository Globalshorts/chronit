## 1단계: 회원가입 및 결제

1. 홈페이지에서 원하는 요금제(스타터/프로/마스터) 선택
2. 결제 모달에서 **계좌이체** 진행
3. 입금자명을 **가입 이메일과 동일하게** 입력 (자동 매칭용)
4. 영업일 기준 1일 이내 자동 활성화

## 2단계: 로그인

크로닛 데스크탑 앱을 실행하고 **결제 시 사용한 이메일**로 로그인합니다.
구독 상태가 자동으로 확인되며, 즉시 사용 가능합니다.

## 3단계: 첫 영상 만들기

대시보드에서 새 프로젝트를 생성하면 다음 흐름으로 진행됩니다:

1. **키워드 입력** — 만들고 싶은 주제 입력 (예: "여름 다이어트 식단")
2. **상품 선택** — 쿠파스에서 관련 상품 자동 검색 및 선택
3. **AI 스크립트 생성** — 후킹 카피 + 본문이 한국어로 자동 작성됨
4. **자막 자동 정렬** — 발음/단위까지 완벽 대응
5. **소스 매칭** — 저작권 안전한 영상 소스 자동 연결
6. **완성** — 약 1분 내 영상 1개 출력

## 4단계: 채널 연동 (수익화)

영상 완성 후 자동 수익화를 위해 다음을 연동하세요:

- **쿠파스 (Coupars)** — 상품 검색 + 수익 추적
- **인포크링크 (InforkLink)** — 단축 링크 + 클릭 데이터 분석

> 자세한 연동 방법은 추후 업데이트 예정입니다.

## 5단계: 다중 채널 운영 (Pro / Master)

Pro 이상 플랜은 다중 채널을 한 번에 관리할 수 있습니다.

- 채널별 기획 템플릿 저장
- 일괄 영상 생성 (하루 5~10개)
- 채널 간 콘텐츠 자동 분배
