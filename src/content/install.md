## 1단계: Windows 설치 파일 다운로드

홈페이지 상단의 **Windows 다운로드** 버튼을 클릭하거나, [여기를 클릭](https://github.com/Globalshorts/chronit/releases/latest/download/Chronit_Setup.exe)해서 최신 버전을 받아주세요.

- 파일 크기: 약 513MB
- 지원 OS: Windows 10 / 11

## 2단계: SmartScreen 경고 처리

첫 실행 시 Windows가 **"PC를 보호했습니다"** 경고를 표시합니다.
이는 코드사이닝 인증서가 누적되기 전까지 나타나는 일반적인 경고입니다.

**처리 방법:**
1. 경고창에서 **추가 정보** 클릭
2. 아래에 나타나는 **실행** 버튼 클릭

**① 추가 정보 클릭**

![SmartScreen 1단계](https://oxygqtbdpnxxcgzwdlzi.supabase.co/storage/v1/object/public/manual-images/smartscreen-step1.png)

**② 실행 버튼 클릭**

![SmartScreen 2단계](https://oxygqtbdpnxxcgzwdlzi.supabase.co/storage/v1/object/public/manual-images/smartscreen-step2.png)

> 보안상 문제 없는 정상 경고입니다. 누적 다운로드가 쌓이면 자동으로 사라집니다.

## 3단계: 설치 마법사 진행

설치 마법사가 열리면 아래 순서대로 진행해주세요.

1. **다음** 버튼을 클릭합니다.
2. 설치 경로를 확인하고 **다음** 버튼을 클릭합니다. (기본 경로 권장)
3. 설치가 완료되면 **마침** 버튼을 클릭합니다.

> 설치 후 바탕화면에 크로닛 아이콘이 생성됩니다.
